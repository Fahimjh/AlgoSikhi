Advanced Vector Operations

Beyond the basics, vectors offer advanced operations that give us more control and flexibility when working with dynamic arrays.

ðŸ”‘ Key Operations & Their Complexities:

  • insert(position, value): Inserts a value at the given position in the vector.
    Use: Add elements anywhere, not just at the end.
    Time Complexity: O(n)
  • erase(position): Removes the element at the given position.
    Use: Delete elements from any position.
    Time Complexity: O(n)
  • begin() & end(): Get iterators to the start and end of the vector.
    Use: Useful for looping through the vector.
    Time Complexity: O(1)
  • rbegin() & rend(): Get reverse iterators to the vector.
    Use: Loop through the vector backwards.
    Time Complexity: O(1)
  • assign(count, value): Assigns new values to the vector, replacing old ones.
    Use: Quickly fill or reset the vector.
    Time Complexity: O(n)
  • swap(v1, v2): Swaps the contents of two vectors.
    Use: Exchange data between vectors efficiently.
    Time Complexity: O(1)
  • Iteration using iterators: Traverse the vector using iterators.
    Use: Access or modify elements in a loop.
    Time Complexity: O(n)

These operations make vectors even more powerful for advanced data manipulation.
