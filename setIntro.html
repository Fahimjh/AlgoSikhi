Introduction to set

A set is a data structure that stores a collection of unique elements. It does not allow duplicate values and usually stores elements in a sorted or unordered way depending on the type of set. Sets are helpful when you only care about whether a value exists, not how many times.

  • Set: A set stores unique elements in sorted order (ascending by default). No duplicates are allowed.
  • unordered_set: Similar to set, but stores elements in no particular order. Also does not allow duplicates.
  • multiset: A multiset allows duplicate elements and stores them in sorted order.
