Array Update (Insert, Delete)

These operations are easiest compared to other operations for array data structure. There are two types of Insertion and Deletion. Insert/Delete at last position and Insert/Delete at any other than last position.

Insertion:

  • To insert one/couple of elements at last position, first we need to check if there are space available in the array for that/those elements.
  • If space isn't available, then the new value overwrites the old value (i.e. array[i]=element.)
  • If space available, then just put the elements in the last position (i.e. array[i]=element.) if its for one element
  • If there are multiple elements to insert at last, then we add elements with a loop which starts from last element and will end at last + no. of elements and will put elements in the array sequetially.
  • If the insertion is not at last and space available for insertions then, we need to shift elements to the right.
  • For one element to insert, shift every element to the right from desired index to (last + 1) index.
  • For multiple elements to insert, shift every element to right from (desired + no. of elements) index to (last + no. of elements) index.

Deletion:

  • To delete one/couple of elements at last position, we just reduce the array size, because array index couldn't be empty. There must be value from user or it will be occupied by garbage value.
  • If one element needs to delete, then just reduce array size by 1 (i.e. array[5] to array[4]).
  • If multiple elements need to delete from last, then reduce array size by no. of elements (i.e. array[5] to array[2]).
  • If the deletion is not at last then, we need to shift elements to the left.
  • For one element to delete, shift every element to left from (desired +1) index to last index.
  • For multiple elements to delete, shift every elements to left from (desired + no. of elements) index to last index.
