Sorting Vector

We've learned some sorting algorithms for arrays in the previous lecture. Those algorithms can also be used to sort vectors. Since vectors allocate contiguous memory locations like arrays, the time and space complexity for those algorithms remain the same as for vectors like arrays. So we aren't going to learn those algorithms again. To sort vectors, Most programming languages provide a built-in sort function to quickly arrange the elements of a vector in ascending or descending order. This function is easy to use and efficient for most cases.

  • Much faster and simpler than writing own sorting algorithm. Time complexity O(n log n), space complexity O(n).
  • Built-in sort functions are widely used for their simplicity and speed, making them a great choice for most sorting tasks.
