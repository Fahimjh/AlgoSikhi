Array Search Algorithms

There are two common search algorithms which are used most. They are-

Linear Search:

  • Simplest searching algorithm which checks the whole array from one end to another.
  • If the algorithm finds desired element, it returns its index; otherwise, it returns -1.
  • The time complexity of this algorithm is O(N) where 'N' is the length of the array, as it traverse the whole array.
  • This search algorithm doesn't need a sorted array
  • Easy to understand but takes more time for O(N) complexity.

Binary Search:

  • Efficient search algorithm for finding an element in a sorted array
  • Uses divide and conquer technique- repeatedly divides the search interval in half.
  • For that technique, it eliminates half of the elements in each step.
  • The time complexity of this algorithm is O(log N) where 'N' is the length of the array.
  • It is much faster than Linear Search for large datasets
